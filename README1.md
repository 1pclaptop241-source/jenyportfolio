# Jeny Sam - Geophysics Portfolio

This is a static portfolio website building with HTML5 and CSS3.

## How to Edit
- **Text**: Open `index.html` in any text editor (Notepad, VS Code) to change text.
- **Styles**: Open `styles.css` to change colors or fonts.
- **Photo**: Replace `assets/Jenyphoto.jpg` with a new photo (keep the same name) or update the `src` in `index.html`.

## How to Host on GitHub

1.  **Create a Repository**: Go to GitHub and create a new repository (e.g., `my-portfolio`).
2.  **Upload Files**: Upload the entire contents of this folder (`index.html`, `styles.css`, and the `assets` folder) to your repository.
3.  **Enable GitHub Pages**:
    *   Go to your repository **Settings**.
    *   Click on **Pages** in the left sidebar.
    *   Under **Build and deployment**, select **Source** as `Deploy from a branch`.
    *   Select `main` (or `master`) branch and folder `/ (root)`.
    *   Click **Save**.
4.  **Visit**: After a few minutes, your site will be live at `https://<your-username>.github.io/<repository-name>/`.
